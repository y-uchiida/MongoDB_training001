# データベースの操作

1. sample データベースに接続  
   データベースが空であれば、操作対象を変更するのみで実際にデータベースは作られない

```js
use sample
```

2. 接続中のデータベースの内容を確認

```js
db;
```

3. データベースの一覧を表示  
   sample データベースが空の場合は一覧に表示されない

```js
show dbs
```

4. データベースにコレクション(RDB でいうところのテーブル)`test` を追加
   このあと、再度データベース一覧を確認すると、`sample` が表示される

```js
db.createCollection("test");
```

5. データベース削除

```js
db.dropDatabase();
```
